Medical Imaging Input

Radiomics begins with standardized medical images — CT, MRI, or PET/CT. Pixel intensities encode tissue properties: Hounsfield Units (CT), T1/T2 relaxation (MRI), SUV (PET). Preprocessing includes resampling to isotropic voxels, intensity normalization, and optional augmentation.

Tumor Segmentation

A deep learning segmentation model (3D U-Net, nnU-Net) delineates the Region of Interest (ROI). The segmentation mask defines exactly which voxels are used for radiomic feature extraction — this step is critical since feature values are extremely sensitive to mask boundaries.

Deep Learning Feature Extraction

A pretrained CNN (ResNet, EfficientNet, or task-specific 3D CNN) extracts high-dimensional feature representations from the ROI. Feature maps from multiple layers encode different abstraction levels. These deep features complement — or replace — handcrafted radiomic features.

Radiomic Feature Quantification

1,000+ handcrafted features are extracted from the segmented ROI across 5 IBSI-defined categories. Features capture shape, intensity statistics, texture (GLCM, GLRLM, GLSZM), and wavelet-decomposed variants. DL features and handcrafted features are combined or compared.

Prediction Model

Selected radiomic + deep features feed into a ML/DL classifier or regressor. Common approaches include LASSO-regularized Cox regression for survival, random forests with SHAP explanations, or end-to-end deep networks with attention mechanisms over the feature set.

Clinical Outcome

The final model outputs a clinical prediction — tumor grade, treatment response (RECIST), OS/PFS survival probability, or recurrence risk. Performance is validated on held-out data with AUC, C-index, and calibration curves. Explainability via SHAP or GradCAM closes the loop.
